# CoffeeOrderingSystem

CoffeeOrderingSystem is a Python-based console application that simulates the basic workflow of a coffee shop. Users can browse the available menu, customize their drinks by choosing sizes and toppings, and receive a final invoice showing the total cost.

## Overview
This program demonstrates fundamental programming concepts such as:
- Menu-driven user interaction
- Input validation
- Conditional logic
- Loop-based order processing

## Main Features
- Interactive text-based menu system
- Multiple coffee options with size-based pricing
- Includes up to three complimentary toppings per drink
- Automatically calculates additional topping charges
- Generates a clear invoice with order details and total cost
